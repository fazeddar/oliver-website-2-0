browserfetch

a neofetch-style browser toolkit — as a bookmarklet

Install

Drag the button below onto your bookmarks bar. It launches browserfetch from your local server so the AI tab can talk to Ollama.

⊞ browserfetch

Cannot see the bar? Press Ctrl+Shift+B (or Cmd+Shift+B).
Hosted helper URL: https://tools.oliverstools.bigguy8014.workers.dev/browserfetch.html
The bookmarklet opens a separate draggable tool window and passes the current page URL, title, and selected text into it.

Tabs
⊞ fetch — neofetch-style system info ✎ notes — persistent notepad 🗄 storage — localStorage / cookies ⚙ tools — JSON / Base64 / URL encode 🤖 ai — browser AI or local endpoint ⍁ network — connection & page info
